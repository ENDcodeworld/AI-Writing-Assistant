## 🚀 快速开始

### 安装运行

```bash
# 克隆项目
git clone https://github.com/ENDcodeworld/AI-Writing-Assistant.git
cd AI-Writing-Assistant

# 安装依赖
pip install -r requirements.txt

# 运行应用
python main.py
```

### 打包APK（Android）

```bash
# 安装buildozer
pip install buildozer

# 初始化配置
buildozer init

# 打包APK
buildozer android debug

# 生成的APK在 bin/ 目录下
```

## 📖 使用指南

### 1. 文章生成
1. 选择"文章生成"功能
2. 选择风格（科技/商业/生活）
3. 输入主题关键词
4. 点击"开始生成"

### 2. 营销文案
1. 选择"营销文案"功能
2. 选择类型（产品/服务/活动）
3. 输入产品名称
4. 点击"开始生成"

### 3. 论文助手
1. 选择"论文助手"功能
2. 选择学科领域
3. 输入论文主题
4. 点击"开始生成"

### 4. 诗歌创作
1. 选择"诗歌创作"功能
2. 选择诗歌类型
3. 输入主题或情感
4. 点击"开始生成"
